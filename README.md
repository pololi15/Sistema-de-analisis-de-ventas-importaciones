#Sales & Imports Analytics

Sistema ETL para analizar ventas e importaciones usando Python, pandas, PostgreSQL y dashboards.

## Objetivo

Construir un pipeline que permita:

- Importat archivos CSV/Excel
- Limpiar y homologar datos.
- Detectar incosistencias.
- Generar KPIs empresariales.
- Visualizar resultados en un dashboard.

## Stack inicial

- Python
- Pandas
- PostgreSQL
- AQLAlchemy
- FastAPI
- Docker
- Metabase / Power BI
